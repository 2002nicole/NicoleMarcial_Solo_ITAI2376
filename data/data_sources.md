# Data Sources

## Recipe Dataset
This project uses a filtered cookie-focused recipe dataset derived from Food.com recipes through Kaggle.

Main source used:
- Food.com Recipes with Ingredients and Tags (Kaggle)  
  https://www.kaggle.com/datasets/realalexanderwei/food-com-recipes-with-ingredients-and-tags

The full raw dataset is not redistributed in this repository. This `data/` folder includes only a small sample file used to illustrate the retrieval setup for the project.
